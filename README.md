# <span style="color:orange">Pro</span><span style="color:lightblue">Desk<span>

Projeto de API - **5º Semestre (2026)** da Fatec São José dos Campos -
**Bug Busters**

O objetivo deste projeto é desenvolver uma plataforma mobile de Help Desk para serviços de TI, permitindo o registro, acompanhamento e gerenciamento de chamados técnicos. A aplicação busca centralizar as solicitações de suporte, facilitando a comunicação entre usuários e equipe de TI e proporcionando maior organização, agilidade e controle no atendimento de demandas tecnológicas.
